================================================================
 新崎川ロガー → Claude Code への引っ越し  はじめにお読みください
================================================================

やることは3つだけです。

【1】 この ZIP を分かりやすい場所に解凍する
      （いつもの「noyama」など、見つけやすいフォルダでOK）
      解凍すると「niizaki-water-log」フォルダができます。
      中に .github / data などのフォルダも入っています（そのままで大丈夫）。

【2】 そのフォルダで Claude Code を開く
      ・ターミナルを開く
      ・解凍したフォルダに移動する  例:  cd ~/noyama/niizaki-water-log
      ・claude と打って Enter

【3】 下の枠の中の文を、そのまま Claude Code に貼り付ける
----------------------------------------------------------------
このフォルダにある新崎川の水位・雨量ロガーを、GitHub Actions で
1時間ごとに自動実行されるよう、公開（Public）リポジトリにセットアップ
してください。私は GitHub 初心者です。各ステップをやさしく日本語で
説明しながら進めてください。git や gh（GitHub CLI）が未インストール／
未ログインなら、まずその確認と案内からお願いします。最後にワークフローを
手動で1回実行して、data/ の CSV が更新されることまで確認してください。
----------------------------------------------------------------

これだけで、あとは Claude Code が
「リポジトリ作成 → アップロード → 権限設定 → 初回実行 → 動作確認」
までガイドしながら進めてくれます。

※ 詳しい中身を知りたいときは README.md を見てください。
※ Claude Code の導入（インストール）からサポートが必要な場合は、
　 元のチャットでその旨を伝えてください。
